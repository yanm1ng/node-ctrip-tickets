# node-ctrip-tickets

携程火车票查询，查询有票的列车

## 环境
* 安装 `node.js`

## 运行
```
$ npm install
```

修改 `main.js` 中的 `DepartureCity`, `ArrivalCity`, `DepartureCityName`, `ArrivalCityName` 四个参数

```
$ node main.js 2017-01-26
```
